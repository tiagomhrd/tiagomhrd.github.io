Journals

A second-order penalty-based node-to-segment contact using the Virtual Element Method

  • DOI: https://doi.org/10.1016/j.finel.2024.104183
  • Journal: Finite Elements in Analysis and Design, Volume 237, 15 September 2024, 104183.
  • Authors: Tiago Fernandes Moherdaui, Alfredo Gay Neto, Peter Wriggers.
  • Highlights:
    • VEM and NTS are combined in a convenient way to compute elasticity models with contact.
    • VEM’s projection is able to filter the stress oscillations stemming from inconsistent NTS formulation for surfaces.
    • Improvements in stress results are obtained for both straight and curved contact-boundary problems.

This paper shows one of the main findings of my masters studies.
The second-order penalty-based Node-to-Segment approach to 2D contact leads to artificial oscillations in the stress results. These can be significantly reduced by the use of Virtual Elements in the adjacent continuum. This does not rid the oscillations completely, but basically avoids a post-processing step which comes naturally from the method's use of projections.

Flexible polyhedra modeled by the virtual element method in a discrete element context

  • DOI: https://doi.org/10.1016/j.cma.2021.114163
  • Journal: Computer Methods in Applied Mechanics and Engineering, Volume 387, 15 December 2021, 114163.
  • Authors: Alfredo Gay Neto, Blaž Hudobivnik, Tiago Fernandes Moherdaui, Peter Wriggers.
  • Highlights:
    • A new method to model flexible polyhedral particles within DEM.
    • Spatial discretization of particles using the Virtual Element Method.
    • Single element per particle scheme.
    • Master–master technique and its degenerations to handle contact.

This paper presents a very promising approach to model flexible particles in more complex shapes (polyhedra, in this case).
The presented approach uses the Discrete Element Method for the particle framework, but employs the Virtual Element Method (VEM) to account for their flexibility.
The versatility of element shapes that the VEM provides, allows this one element per particle discretization, which allows a more computationally efficient approach compared with coarse finite element meshes per particle.
This also shows the robustness of the master-master contact approach.

Public Code Projects

Here are some programming projects that I have decided to make public.
My language of preference is C++.

Mnl - Monomial indexing framework

A generalized framework for indexing monomials by integers and extracting useful information from these indices.
This works for polynomial spaces of any order and over any number of variables.
A simple framework for working with polynomials is provided, along with some quadrature rules for 1D domains.

Ptp - Polytope domain integration

A framework for computing monomial integrals over polytopes (so far restricted to polygons and polyhedra).

VES - Virtual Element Spaces

A library for producing the relevant projectors for different formulations of the Virtual Element Method.
Still a work in progress, the idea is to provide the basis for anyone to implement the method for different classes of problems with the projector assembly procedure.
